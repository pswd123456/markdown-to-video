**RAG Practice** 是一个致力于将 RAG 理论转化为生产级代码的全栈工程实践项目。不同于简单的 Demo，本项目深入探索了从数据处理到检索优化的实际落地问题。

**核心特性包括：**

- **深度数据处理**：采用 **Docling** 解析复杂 PDF 文档，保留层级结构并进行精细化的语义切片。
    
- **高精检索策略**：基于 Elasticsearch 实现 **Hybrid Search**（向量 + BM25 关键词），结合 **RRF** 融合排序与 **TEI** 二次重排序（Rerank），显著提升召回准确率。
    
- **闭环评估体系**：集成 **Ragas** 与 **Langfuse**，构建了从“自动生成测试集”到“量化指标评估”及“全链路 Trace 监控”的自动化闭环。
    
- **扎实工程架构**：基于 FastAPI + Next.js 构建，实现了异步任务队列（Redis + Arq）、父子索引策略、权限管理及 Docker 一键容器化部署。
    

该项目适合希望深入理解 RAG 架构、工程化落地及效果评估的开发者参考与实践。
